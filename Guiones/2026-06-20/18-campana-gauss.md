# 18 — La Figura que Repite la Naturaleza

**Gancho (0-5s):** Mides la altura de 1000 personas. Casi todas están cerca del promedio. Muy pocas son muy altas o muy bajas.

**Desarrollo (5-20s):** Las notas de un examen. El peso de las manzanas. Los errores en una fábrica. Todos dibujan la misma figura.

**Clímax (20-40s):** Se llama campana de Gauss. Aparece sola, siempre que algo se repite muchas veces. La naturaleza la usa a cada rato.

**CTA (40-60s):** Guarda esto para cuando veas una gráfica de resultados. Si no parece campana, algo raro pasa. Sígueme.
