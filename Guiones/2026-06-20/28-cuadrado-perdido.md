# 28 — El Cuadrado que Desaparece

**Gancho (0-5s):** Un triángulo se corta en 4 piezas. Se reordenan. Misma forma. Pero falta un cuadrado.

**Desarrollo (5-20s):** Las piezas parecen encajar perfecto. Pero las pendientes no son iguales. No es un triángulo recto. Es una trampa visual.

**Clímax (20-40s):** La hipotenusa no es recta. Tiene una curva diminuta que no ves. Esa curva esconde el área del cuadrado perdido.

**CTA (40-60s):** Guarda esto para cuando alguien diga que las mates son aburridas. Los números pueden hacer desaparecer cosas. Sígueme.
