# Configuración de Agente — Creación de Guiones para Reels

## Regla fundamental

Siempre que se te pida crear, escribir, revisar o proponer un **guion para un reel** (Instagram, TikTok, YouTube Shorts), **debes leer y aplicar obligatoriamente** el archivo:

> **`REELS_GOLDEN_RULES.md`**

Ese archivo contiene el framework completo validado con 700+ clientes, incluyendo:

- Las reglas de oro del hook (80% del éxito del video)
- El framework TCM (Tropicalizar, Comprender, Maximizar)
- Los 5 tipos de hooks que funcionan
- La estructura del guion híbrido (gancho → promesa → pico de interés → valor → CTA)
- Checklist de validación pre-publicación
- Estrategia de escalado con anuncios

## Proceso obligatorio antes de escribir un guion

1. Lee `REELS_GOLDEN_RULES.md` completo
2. Identifica el nicho y audiencia objetivo
3. Elige el **tipo de hook** (curioso, negativo, dolor, transformación, error común)
4. Aplica la **estructura del guion híbrido**
5. Valida contra el **checklist de pre-publicación**
6. Asegura que el contenido sea **aplicable** (que el espectador pueda guardarlo/usarlo)
7. Verifica que el **CTA tenga una razón convincente**

## Idioma

Todos los guiones deben generarse en **español neutro latinoamericano**. Sin modismos locales de un país en particular (ni voseo rioplatense, ni tuteo mexicano exclusivo, ni español de España). Usar términos que se entiendan en toda Latinoamérica.

## Base de Hooks Reales (Inspiración Validada)

Siempre que crees un guion, **debes consultar y usar como referencia** el archivo:

> **`hooks_consolidado.md`**

Ese archivo contiene **+1,900 hooks de reels virales reales** organizados por sector. Su uso es obligatorio como:

1. **Inspiración de estructura** — toma hooks que ya funcionaron y adáptalos al tema/cliente
2. **Benchmark de métricas** — contrasta las vistas esperadas contra reales
3. **Banco de patrones** — identifica qué tipo de hook (curioso, negativo, dolor, transformación, error común) funciona mejor en cada nicho
4. **Referencia de tono y estilo** — copia el ritmo, la longitud y la voz de hooks probados

**Regla:** No inventes hooks desde cero sin antes revisar `hooks_consolidado.md` y adaptar al menos un patrón existente.

## Recordatorio

No importa el nicho (fitness, finanzas, recetas, real estate, salud, marketing, etc.). El framework es universal. Adáptalo siempre al cliente y audiencia específica.
